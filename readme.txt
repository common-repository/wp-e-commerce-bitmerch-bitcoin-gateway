Adds BitMerch bitcoin payment gateway to the WP e-Commerce plugin.


== Description ==

This plugin allows to accept bitcoin currency in your store. Payments processed by bitmerch.com gateway.

= Introductory Video =


[Documentation](https://bitmerch.com/pages/api) |


== Installation ==

= Minimum Requirements =

*   WP E-Commerce 3.8+
*   WordPress 3.3+
*   PHP 5+

= Install Instructions =

1. Copy this directory contents into your WordPress install path.
2. Enable API access in your BitMerch account
3. Enable bitcoin payment method in "payments" tab of wp-e-commerce plugin settings
4. Copy "Merchant ID" and "API Secure Key" from bitmerch to the wordpress payment method settings
5. Done.
